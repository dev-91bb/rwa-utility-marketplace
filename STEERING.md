# RWA Project Steering

## Table Formatting

Use markdown tables with proper column alignment:

```
| Column1   | Column2     | Column3 |
|-----------|-------------|---------|
| Value1    | Description | ✅      |
| Value2    | Description | ❌      |
```

- Align columns with consistent spacing
- Use ✅ and ❌ for boolean values
- Keep descriptions concise

## Response Style

- Be concise and direct
- Use tables for structured data
- Use code blocks for code/commands
- Skip unnecessary explanations
